Welcome to the Refactored Website

This page demonstrates the new modular template system that has been implemented across the entire website. All pages now use:

  • Dynamic header and footer templates loaded via JavaScript
  • Global CSS for consistent styling across all pages
  • Centralized JavaScript for login state and utilities
  • SEO-optimized meta tags on every page
  • Lazy loading for images to improve performance
  • Accessibility features like ARIA labels and semantic HTML

How It Works

When a page loads, the main.js script automatically:

  1. Fetches templates/header.html and injects it into #app-header
  2. Fetches templates/footer.html and injects it into #app-footer
  3. Checks user login state from localStorage
  4. Shows login button or welcome message based on state
  5. Sets up lazy loading for images
// main.js automatically runs this on every page:
async function loadTemplates() {
  const headerResponse = await fetch('templates/header.html');
  const footerResponse = await fetch('templates/footer.html');
  
  document.querySelector('#app-header')
    .innerHTML = await headerResponse.text();
  document.querySelector('#app-footer')
    .innerHTML = await footerResponse.text();
}

Images with Lazy Loading

All images on this page use lazy loading. They will only load when visible in the viewport:

This improves page load time significantly, especially on pages with many images.

Login State Management

The login state is stored in the browser's localStorage and managed by main.js. Users can:

Login: Click the "Login" button in the header
View Status: Your login status is shown in the top right of the header
Logout: Click "Logout" to clear your session

This is useful for showing personalized content to logged-in users.
